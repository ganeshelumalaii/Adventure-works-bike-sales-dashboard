# Adventure-works-bike-sales-dashboard
Data Analysis project using power bi

Data Analytics Project – Sales Performance Dashboard Overview

This project demonstrates an end-to-end data analytics workflow, starting from loading and analyzing a CSV dataset to building an interactive Power BI dashboard and presenting insights through a professional report and presentation.

The objective of this project is to transform raw data into meaningful business insights that support data-driven decision-making.

Dataset
Source: CSV File
Type: Sales Data
Contents: Product details, sales transactions, customer information, quantities, revenue, and returns data.
Purpose: To analyze sales performance, customer behavior, product trends, and key business metrics.
Tools Used
Tool	Purpose
Microsoft Excel	Data review and preprocessing
CSV Dataset	Raw data source
Power BI	Data modeling, visualization, and dashboard creation
Microsoft Word	Project report documentation
Gamma	Presentation (PPT) creation
DAX	Calculated measures and KPIs in Power BI

Project Steps
1. Data Loading
Imported the dataset from a CSV file.
Verified data quality and structure.
Checked for missing values and inconsistencies.
2. Data Preparation
Cleaned and transformed data where necessary.
Standardized column names and formats.
Prepared the dataset for analysis and visualization.
3. Dashboard Development
Imported cleaned data into Power BI.
Created relationships between tables (if applicable).
Developed DAX measures for key performance indicators (KPIs).
4. Report Creation
Documented project objectives, methodology, analysis, and findings.
Summarized business insights and recommendations.
5. Presentation Development
Created a professional presentation using Gamma.
Highlighted project objectives, dashboard visuals, and key findings.
Dashboard Features

The Power BI dashboard includes:

Total Sales Analysis
Revenue Trends
Product Performance
Return Rate Analysis
Customer Insights
Category-Wise Performance
Interactive Filters and Slicers
KPI Cards and Visualizations
Results & Insights

Key outcomes from the analysis include:

Identified top-performing products and categories.
Analyzed sales trends across different periods.
Measured return rates and product performance.
Discovered opportunities for improving sales and operational efficiency.
Provided actionable insights to support business decision-making.
